Front-end: ReactJs
Back-end: ExpressJs
Database: Firebase
Messaging Appliaction: Twilio

Steps to use the application:
1. Enter your phone number with your country code (eg +84)
2. Click submit phone number
3. The access code will be sent your phone
4. The second form will show up for you to fill in the access code
5. Enter the access code and the system will validate it
